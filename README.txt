Toko Donny - Aplikasi Penjualan Sederhana (UTS)
-----------------------------------------------
Cara pakai:
1. Ekstrak folder 'toko_donny' ke htdocs (XAMPP) atau www (Laragon).
2. Import file 'toko_donny.sql' lewat phpMyAdmin (pastikan user root tanpa password atau sesuaikan koneksi).
3. Buka browser: http://localhost/toko_donny/
4. Semua halaman: index.php, barang.php, pembeli.php, transaksi.php, dashboard.php

Catatan:
- Tidak ada login (akses langsung).
- Gunakan PHP 7.4+/MySQL 5.7+ atau MariaDB. Triggers mungkin perlu izin pada beberapa shared hosting.
